# Multi-Service Network Infrastructure (GNS3)


## 📌 Description
This project aims to deploy and manage multiple network services on Linux with automation using Shell scripts.

The services implemented in this project are:
- NFS (file sharing)
- MySQL Database
- Monitoring (Prometheus / Grafana)
- Web Server (Apache)
